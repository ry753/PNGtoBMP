【プロジェクト概要】
・PNG画像をBMP画像に変換するプログラムです。
・すべてのPNG画像に対応しているわけではなく、以下特徴を持つPNG画像のみに対応しています。
　ビット深度　　：0x08, カラータイプ　：0x06
　圧縮手法　　　：0x00, フィルタタイプ：0x00, インタレース方式：0x00

【動作】
・windows11での動作のみを確認しています。

【公開物説明】
・「exe」と「src」があります。
・「exe」はプログラムの実行のみ行うことを目的としたフォルダです。
・inputフォルダ内の画像を指定して変換し、outputフォルダに保存します。
・「src」はvisualstudio実行用のソースコードです。
・input及びoutputフォルダに同封されている「dummy.txt」は削除しても問題ありません。

【使い方】
・配布物のexeファイルはそのまま使うことができます。
・inputファイルに変換したいpng画像を入れ、exeを起動してファイル名を入力します。
・親フォルダ内にexeファイル、inputフォルダ、outputフォルダを格納
・inputフォルダ、outputフォルダのフォルダ名は「input」「output」にしてください。
・inputフォルダ内に変換したいPNG画像を格納。
・exe起動後、inputフォルダ内の画像名を入力することで変換が行われます。
・処理完了後、outputフォルダに変換画像が生成されています。
　１）visualstudioで空のプロジェクトを作成。
　２）src内のcpp,header,libフォルダのすべての中身をslnと同じ階層に格納。
　３）同じくslnと同じ階層に「input」「output」という空のフォルダを作成。
　４）visualstudioを開き、ソースファイルに.cppファイル、ヘッダーファイルに.hファイル、リソースファイルに.libファイルを追加。
　５）ビルドを行う。
